# 设计说明

## 需求
编写一个图形界面, 用于显示修改mpq文件
1. python + pyside编写界面。按模块分成文件，不要放到同一个文件里
2. 打开mpq文件，左侧用树形结构显示目录树
3. 右侧显示该目录的文件、目录
4. 拖动文件到右侧把文件加到mpq里。
5. 拖动左侧目录或右侧文件和目录，可以解压到指定拖动的目的文件加。
6. 左右侧都支持类型文件夹的操作，可以删除文件/目录，新建目录，修改文件名等等。
7. 打开mpq时可以指定文件列表。
8. 提供压缩功能，以减小mpq体积。
9. 有合理的工具栏，便于快速完成上面的各种操作。

## 规范
1. 新加的功能都需要有响应的测试。在test目录里创建测试脚本
2. 改动完成后，需要保证LSP不报错，test目录里面的测试能通过
